# Online Shopping Cart

The Online Shopping Cart is a Java application that simulates an e-commerce platform's shopping cart functionality. It allows users to add, remove, and view items in their cart, apply discounts, and checkout. This project demonstrates the use of Java Collections Framework, Exception Handling, Streams API, Lambda Expressions, and Concurrency.

## Features

- Add items to the cart
- Remove items from the cart
- View all items in the cart
- Apply discounts to the cart
- Checkout and calculate the total price
- Concurrency support for managing cart operations

## Technologies Used

- Java
- Collections Framework (List, Set, Map)
- Exception Handling (try-catch, custom exceptions)
- Streams API
- Lambda Expressions
- Concurrency (Threads, Executors, synchronized methods)
